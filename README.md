# Checklist ไปเปรโต๊ะลอซู

เว็บ checklist สำหรับแชร์ให้เพื่อนกรอกชื่อ ติ๊กของที่เตรียมแล้ว ดู realtime ว่าใครยังขาดอะไร และเพิ่ม/ลบหมวดหรือรายการได้เอง

## เปิดในเครื่อง

```bash
npm start
```

แล้วเปิด `http://localhost:3000`

## เอาขึ้นเป็นเว็บจริง

วิธีง่ายสุดคือใช้ Render:

1. สร้าง repository บน GitHub แล้วอัปโหลดโฟลเดอร์นี้
2. เข้า Render แล้วเลือก New Web Service
3. เลือก repository นี้
4. ใช้ค่าเดิมจาก `render.yaml`
5. Deploy แล้วเอาลิงก์ที่ได้ส่งให้เพื่อน

หมายเหตุ: ข้อมูลบันทึกอยู่ในไฟล์ `data/checklist.json` เหมาะกับทริปเล็ก ๆ ถ้าต้องการใช้ระยะยาวมาก ๆ ควรเปลี่ยนเป็นฐานข้อมูลจริง
